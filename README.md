# student-fitness-lifestyle-analysis(PYTHON)
Description
This project performs **data analysis and visualization** on a student fitness and lifestyle dataset stored in an Excel file.
Using **Python, Pandas, and Matplotlib**, various questions related to exercise habits, screen time, snack expenditure, and lifestyle patterns are analyzed.
Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Excel (.xlsx dataset)

**Analysis Performed:**
**Q1.** Number of female students exercising **more than 6 and less than 13 hours per week**  
- Histogram and boxplot used for visualization

**Q2.** Most preferred type of exercise among **male students**  
- Bar chart using value counts

**Q3.** Minimum and maximum **daily screen time**, grouped by gender  
- Grouped data and bar chart visualization

**Q4.** **Average and standard deviation** of weekly pocket money spent on snacks  
- Histogram with mean line
- Boxplot for distribution analysis

**Q5.** Number of **male and female students wearing glasses**  
- Grouped bar chart by gender
